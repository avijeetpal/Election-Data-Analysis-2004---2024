# 📊 Election Data Analysis & Visualization (Lok Sabha 2004–2024)

## 📌 Project Overview
This project focuses on analyzing Lok Sabha election data from 2004 to 2024. The objective is to transform raw, unstructured data into meaningful insights using data analysis and visualization techniques.

---

## 🎯 Objectives
- Analyze voter turnout trends over multiple election years  
- Study gender-wise participation (male vs female voters)  
- Examine election expenditure and cost per voter  
- Build an interactive dashboard for data-driven insights  

---

## 📂 Dataset
- **Source:** Election Commission of India (ECI)  
- **Format:** Unstructured PDFs converted into CSV  
- **Coverage:** 2004–2024  
- **Level:** Constituency & State  

---

## 🛠️ Tools & Technologies
- **Python:** Pandas, NumPy, Matplotlib  
- **Power BI:** Dashboard & Visualization  
- **Jupyter Notebook**  
- **Git & GitHub**  

---

## ⚙️ Project Workflow

### 1️⃣ Data Collection
- Collected election data from ECI (PDF format)  
- Converted into structured CSV  

### 2️⃣ Data Cleaning & Transformation
- Handled missing values  
- Standardized formats  
- Fixed inconsistent state names  
- Aggregated constituency data into state-level  

### 3️⃣ Exploratory Data Analysis (EDA)
- Analyzed voter trends  
- Compared male vs female participation  
- Studied election cost trends  

### 4️⃣ Data Visualization
- Built interactive Power BI dashboard  
- Added KPIs, charts, and slicers (Year & State)  

---

## 📊 Key Insights
- 📈 Female participation increased by **~46%** (2004–2024)  
- 💰 Significant increase in government election expenditure  
- 📊 Voter turnout shows steady growth  
- ⚖️ Gender gap is gradually reducing
